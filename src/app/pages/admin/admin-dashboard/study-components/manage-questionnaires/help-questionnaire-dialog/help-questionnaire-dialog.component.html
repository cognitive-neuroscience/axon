Go to survey monkey and navigate to My Surveys and click on the survey you created.
Under Design Survey, click the Logic option on the left side toolbar.

Click the Custom Variables option and add two custom variables.
Make sure that the first variable name you add is "s" (to collect the subject ID) and "e" (to collect the associated study code).
The accompanying labels can be anything you want and will appear on the Excel spreadsheet when you download the responses.
In the below example, the labels are set as subjectID and study code.

Go to Collect Responses and click Add New Collector. Select Web Link Collector.
Copy the URL. This will be the URL that you paste into the sharplab platform. It will look like this:
https://www.surveymonkey.com/r/XK8SD38?s=[s_value]&e=[e_value]
Important! Make sure you set the value in the URL as s=[s_value]&e=[e_value] in this order.
