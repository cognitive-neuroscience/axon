1. Go to pavlovia and navigate to your desired task

2. Copy the URL. It should be of the form https://run.pavlovia.org/Sharp_lab/your-task-name/html

3. Click "Add a new Custom Task" and paste the pavlovia link into the Custom Task URL input.

4. URL Query parameters are automatically appended to the end of the URL. This means that when the participant does the task, data should be included such as their ID and the associated study code.
